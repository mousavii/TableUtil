Tableutil

a JQuery-UI plugin to lock the table header. also lock and unlock a specific column.

View On GitHub
Downloads:
  • ZIP
  • TAR

    Welcome to tableutil Pages.

    This plugin is developed to provide a number of services over a table. these features are:

    • Lock table header
    • Lock a column such a way that you can scroll the table either horizontally or vertically. 

    
    you need to specify the selector which is a table.
    $(selector).TableUtil();
    activate the plugin Activate
    deactivate the plugin Deactivate
